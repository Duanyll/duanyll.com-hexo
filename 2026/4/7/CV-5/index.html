消融实验：删除 模块

实验说明

模块是 ISTA 算法中的梯度下降步，对应代码中 BasicBlock.forward 的以下两行：

这两行实现了 ，其物理含义是将当前估计 沿着数据保真项的负梯度方向移动，使重建结果与 CS 测量值 保持一致。

删除 模块后，网络每个 phase 仅保留 模块（近端映射步），即只做稀疏变换 + 软阈值 + 逆变换，而完全丧失了与观测数据 的联系。此时网络退化为一个纯粹的去噪/稀疏化网络，无法利用测量信息来校正重建方向，预期性能将大幅下降。

对比总结

ISTA-Net+ (完整) ISTA-Net+ (删除 )
梯度下降步 无（跳过）
近端映射步 保留 保留
数据一致性约束 有 无
与测量值 的联系 每个 phase 都利用 仅初始化时使用

实验设置

  • 训练集: 91 张图像中提取的 88,912 个 33x33 图像块
  • 测试集: Set11 (11 张标准测试图像)
  • 训练配置: Adam 优化器，学习率 1e-4，200 epochs，batch size 64，
  • 评估指标: PSNR (dB) 和 SSIM
  • CS 采样率: 1%, 4%, 10%, 25%, 30%, 40%, 50%

PSNR 与 SSIM 随采样率变化趋势

删除 模块后（蓝色虚线），PSNR 在所有采样率下均大幅低于完整模型（约 7~10 dB），且差距随采样率升高而增大。这说明当可用测量信息越多时， 模块对利用这些信息越关键。

删除 模块的 PSNR 下降幅度

模块的移除导致约 7~10 dB 的性能下降，远大于 ISTA-Net+ 相对于 ISTA-Net 的提升（0.1~0.7 dB）。这充分表明 模块是 ISTA-Net 架构中最核心的组件：它提供的数据一致性约束是网络能够高质量重建的根本保证。

逐图对比 (CS Ratio = 25%)

在 25% 采样率下，删除 模块导致所有测试图像的 PSNR 均大幅下降，平均降幅约 10 dB。其中 foreman 图像的绝对 PSNR 仍最高（因其内容较简单），但降幅同样显著。

视觉对比 (CS Ratio = 25%)

从视觉对比可以观察到：

  • 删除 模块后，重建图像出现严重的块效应和模糊
  • 误差增量热力图（第四列）显示误差在整个图像上均匀增加，而非局限于特定区域，说明 模块提供的是全局性的数据一致性修正

结论

消融实验验证了 模块（梯度下降步）在 ISTA-Net 架构中的核心作用：

  1. 性能影响巨大：删除 模块导致 PSNR 下降约 7~10 dB，远超其他组件（如残差域算子 R）带来的增益（0.1~0.7 dB）
  2. 物理意义明确： 模块实现了 ISTA 算法的梯度下降步，确保每次迭代都向着与观测数据一致的方向修正。没有它，网络只能做纯粹的稀疏化，失去了与测量值的联系
  3. 体现了算法展开的价值：ISTA-Net 的核心优势在于将传统优化算法的结构嵌入网络。 模块正是这种”优化启发式结构”的直接体现，是区别于普通黑盒网络的关键所在
