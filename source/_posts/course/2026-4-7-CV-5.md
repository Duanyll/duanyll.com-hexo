## 方法简介

### 问题定义

压缩感知 (Compressive Sensing, CS) 的目标是从少量线性测量 $y = \Phi x \in \mathbb{R}^M$ 中恢复原始信号 $x \in \mathbb{R}^N$（$M \ll N$）。CS 比率定义为 $M/N$，比率越低重建越困难。传统方法通过求解带稀疏正则化的优化问题来重建信号：

$$\min_x \frac{1}{2}\|\Phi x - y\|_2^2 + \lambda \|\Psi x\|_1$$

### ISTA-Net

ISTA-Net 将经典的迭代收缩阈值算法 (ISTA) 展开 (unroll) 为深度网络。网络由 $N_p = 9$ 个 phase 组成，每个 phase 对应 ISTA 的一次迭代，包含两个模块：

- **$r^{(k)}$ 模块**（梯度下降步）：沿数据保真项 $\frac{1}{2}\|\Phi x - y\|_2^2$ 的梯度方向更新，确保重建结果与观测数据保持一致：

$$r^{(k)} = x^{(k-1)} - \rho^{(k)} \Phi^\top(\Phi x^{(k-1)} - y)$$

- **$x^{(k)}$ 模块**（近端映射步）：通过可学习的非线性稀疏变换 $F^{(k)}$ 和软阈值操作实现正则化：

$$x^{(k)} = \tilde{F}^{(k)}(\text{soft}(F^{(k)}(r^{(k)}), \theta^{(k)}))$$

其中 $F^{(k)}$ 由两层 3x3 卷积 + ReLU 构成，$\tilde{F}^{(k)}$ 是其结构对称的左逆（满足 $\tilde{F}^{(k)} \circ F^{(k)} = I$），$\theta^{(k)}$ 是可学习的软阈值。所有参数端到端学习。

### ISTA-Net+

ISTA-Net+ 在 ISTA-Net 基础上引入残差域操作算子 $R = G \circ D$，将稀疏变换分解为 $F = H \circ D$，更新公式变为：

$$x^{(k)} = r^{(k)} + G^{(k)}(\tilde{H}^{(k)}(\text{soft}(H^{(k)}(D^{(k)}(r^{(k)})), \theta^{(k)})))$$

利用自然图像残差更可压缩的先验知识，并通过残差连接 (skip connection) 加速训练收敛。

### 损失函数

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{discrepancy}} + \gamma \mathcal{L}_{\text{constraint}}$$

- $\mathcal{L}_{\text{discrepancy}}$: 重建图像与原始图像的 MSE
- $\mathcal{L}_{\text{constraint}}$: 对称约束损失，确保 $\tilde{F}^{(k)} \circ F^{(k)} \approx I$
- $\gamma = 0.01$

## 消融实验：删除 $r^{(k)}$ 模块

### 实验说明

$r^{(k)}$ 模块是 ISTA 算法中的**梯度下降步**，对应代码中 `BasicBlock.forward` 的以下两行：

```python
x = x - self.lambda_step * torch.mm(x, PhiTPhi)
x = x + self.lambda_step * PhiTb
```

这两行实现了 $r^{(k)} = x^{(k-1)} - \rho^{(k)} \Phi^\top(\Phi x^{(k-1)} - y)$，其物理含义是将当前估计 $x^{(k-1)}$ 沿着数据保真项的负梯度方向移动，使重建结果与 CS 测量值 $y$ 保持一致。

**删除 $r^{(k)}$ 模块后**，网络每个 phase 仅保留 $x^{(k)}$ 模块（近端映射步），即只做稀疏变换 + 软阈值 + 逆变换，而**完全丧失了与观测数据 $y$ 的联系**。此时网络退化为一个纯粹的去噪/稀疏化网络，无法利用测量信息来校正重建方向，预期性能将大幅下降。

### 对比总结

|  | ISTA-Net+ (完整) | ISTA-Net+ (删除 $r^{(k)}$) |
|---|---|---|
| 梯度下降步 | $r^{(k)} = x^{(k-1)} - \rho \Phi^\top(\Phi x^{(k-1)} - y)$ | 无（跳过） |
| 近端映射步 | 保留 | 保留 |
| 数据一致性约束 | 有 | **无** |
| 与测量值 $y$ 的联系 | 每个 phase 都利用 $y$ | 仅初始化时使用 $y$ |

### 实验设置

- **训练集**: 91 张图像中提取的 88,912 个 33x33 图像块
- **测试集**: Set11 (11 张标准测试图像)
- **训练配置**: Adam 优化器，学习率 1e-4，200 epochs，batch size 64，$N_p = 9$
- **评估指标**: PSNR (dB) 和 SSIM
- **CS 采样率**: 1%, 4%, 10%, 25%, 30%, 40%, 50%

## 结论

消融实验验证了 $r^{(k)}$ 模块（梯度下降步）在 ISTA-Net 架构中的核心作用：

1. **性能影响巨大**：删除 $r^{(k)}$ 模块导致 PSNR 下降约 7~10 dB，远超其他组件（如残差域算子 R）带来的增益（0.1~0.7 dB）
2. **物理意义明确**：$r^{(k)}$ 模块实现了 ISTA 算法的梯度下降步，确保每次迭代都向着与观测数据一致的方向修正。没有它，网络只能做纯粹的稀疏化，失去了与测量值的联系
3. **体现了算法展开的价值**：ISTA-Net 的核心优势在于将传统优化算法的结构嵌入网络。$r^{(k)}$ 模块正是这种"优化启发式结构"的直接体现，是区别于普通黑盒网络的关键所在
