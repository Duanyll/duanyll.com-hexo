## 方法简介

### 问题定义

压缩感知 (Compressive Sensing, CS) 的目标是从少量线性测量 $y = \Phi x \in \mathbb{R}^M$ 中恢复原始信号 $x \in \mathbb{R}^N$（$M \ll N$）。CS 比率定义为 $M/N$，比率越低重建越困难。

### ISTA-Net

ISTA-Net 将经典的迭代收缩阈值算法 (ISTA) 展开为深度网络。网络由 $N_p$ 个 phase 组成，每个 phase 对应 ISTA 的一次迭代，包含两个模块：

- **$r^{(k)}$ 模块**（梯度下降步）：$r^{(k)} = x^{(k-1)} - \rho^{(k)} \Phi^\top(\Phi x^{(k-1)} - y)$
- **$x^{(k)}$ 模块**（近端映射步）：$x^{(k)} = \tilde{F}^{(k)}(\text{soft}(F^{(k)}(r^{(k)}), \theta^{(k)}))$

其中 $F^{(k)}$ 是由两层卷积 + ReLU 构成的可学习非线性稀疏变换，$\tilde{F}^{(k)}$ 是其左逆（结构对称），$\theta^{(k)}$ 是可学习的软阈值参数。

### ISTA-Net+

ISTA-Net+ 在 ISTA-Net 的基础上引入了**残差域操作**。其核心改进是定义了残差提取算子 $R = G \circ D$，其中：

- $D^{(k)}$: 线性卷积 (1 $\to$ 32 通道)，将图像映射到特征域
- $G^{(k)}$: 线性卷积 (32 $\to$ 1 通道)，将特征映射回图像域

ISTA-Net+ 将稀疏变换分解为 $F = H \circ D$，更新公式变为：

$$x^{(k)} = r^{(k)} + G^{(k)}(\tilde{H}^{(k)}(\text{soft}(H^{(k)}(D^{(k)}(r^{(k)})), \theta^{(k)})))$$

与 ISTA-Net 相比，ISTA-Net+ 增加了残差连接（skip connection），在残差域上进行稀疏化，利用了自然图像残差更可压缩的先验知识。

### 损失函数

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{discrepancy}} + \gamma \mathcal{L}_{\text{constraint}}$$

- $\mathcal{L}_{\text{discrepancy}}$: 重建图像与原始图像的 MSE
- $\mathcal{L}_{\text{constraint}}$: 对称约束损失，确保反向变换是前向变换的近似逆
- $\gamma = 0.01$

## 消融实验："删除 R 模块"

### 实验说明

R 模块即 ISTA-Net+ 中的 $R = G \circ D$ 算子，对应代码中的 `conv_D` (32x1x3x3) 和 `conv_G` (1x32x3x3) 两个卷积层以及残差连接 `x_pred = x_input + x_G`。

**删除 R 模块后，ISTA-Net+ 退化为 ISTA-Net**，具体差异为：

| | ISTA-Net (无 R 模块) | ISTA-Net+ (有 R 模块) |
|---|---|---|
| 稀疏变换输入 | 图像直接输入 (1 ch) | 经 D 映射到特征域 (32 ch) |
| 稀疏变换输出 | 直接输出图像 (1 ch) | 经 G 映射回图像域 (1 ch) |
| 残差连接 | 无 | 有 ($x^{(k)} = r^{(k)} + G(\cdot)$) |
| 操作域 | 图像域 | 残差域 |

### 实验设置

- **训练集**: 91 张图像中提取的 88,912 个 33x33 图像块
- **测试集**: Set11 (11 张标准测试图像)
- **训练配置**: Adam 优化器，学习率 1e-4，200 epochs，batch size 64，phase 数 $N_p = 9$
- **评估指标**: PSNR (峰值信噪比, dB) 和 SSIM (结构相似性)
- **CS 采样率**: 1%, 4%, 10%, 25%, 30%, 40%, 50%

两个模型分别在 7 个采样率下独立训练和测试。

### PSNR 与 SSIM 随采样率变化趋势

![PSNR vs CS Ratio](https://img.duanyll.com/img/dfe384a9.png)

![SSIM vs CS Ratio](https://img.duanyll.com/img/34614a8d.png)

ISTA-Net+ 在所有采样率下均优于 ISTA-Net，且随着采样率的增加，两者差距逐渐增大。

### R 模块带来的 PSNR 增益

![PSNR Gain](https://img.duanyll.com/img/ee8c47d3.png)

R 模块（残差域操作 + 残差连接）带来的 PSNR 增益在 0.09~0.65 dB 之间。增益随采样率升高而增大，说明当可用信息越充分时，残差域的稀疏化优势越明显。在极低采样率 (1%) 下，信息严重不足，R 模块的增益有限。

### 逐图对比 (CS Ratio = 25%)

![Per-Image PSNR](https://img.duanyll.com/img/c8c0b699.png)

在 25% 采样率下，R 模块对所有 11 张测试图均有提升。增益最大的是 barbara (+1.39 dB)，该图包含丰富的规则纹理，残差域稀疏化对此类结构尤为有效。

### 视觉对比 (CS Ratio = 25%)

![Visual Comparison](https://img.duanyll.com/img/ca42c0f6.png)

从裁剪区域的对比可以看出：

- **Monarch**: ISTA-Net+ 在蝴蝶翅膀的斑点纹理上恢复了更多细节
- **barbara**: 头巾条纹区域差异显著，ISTA-Net+ 的条纹更清晰连续
- **cameraman**: 两者视觉差异较小，但 ISTA-Net+ 在三脚架边缘更锐利

差异热力图显示，两个模型的差异主要集中在高频纹理和边缘区域，这与 R 模块在残差域捕获高频细节的设计目标一致。

## 结论

消融实验验证了 R 模块 ($R = G \circ D$) 对 ISTA-Net+ 性能的贡献：

1. R 模块在所有采样率下均提升了重建质量，PSNR 增益 0.09~0.65 dB
2. 增益主要来自残差域稀疏化和残差连接两个因素
3. 高频纹理丰富的图像（如 barbara）受益最大
4. 采样率越高，R 模块的优势越显著
