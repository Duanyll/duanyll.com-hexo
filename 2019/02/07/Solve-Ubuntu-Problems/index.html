duanyll
2019-02-07
技术
Ubuntu

无法关机,一直闪光标

打开/etc/default/grub,删去里面的quiet splash,然后运行sudo update-grub,就可以在开关机时显示日志信息了.

根据我的日志信息,我发现无法关机的原因是Grub在等待运行的程序90秒再强制结束,所以解决方案是关机前先关掉所有打开的程序来避免90秒等待.
